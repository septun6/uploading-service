# uploading-service

## Installation

### Clone rep

```bash
git clone https://github.com/septun6/uploading-service
```

### Install dependencies
```cd api-gateway
npm install
cd ../core-service
npm install
```

## Running
1. From uploading-service
```docker-compose up```
2. From api-gateway in new terminal window
```npm run start```
3. From core-service in new terminal window
4. ```npm run start```
